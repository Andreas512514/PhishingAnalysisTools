## Google Admin Toolbox
![Google Admin Toolbox](https://github.com/Andreas512514/PhishingAnalysisTools/blob/main/Screenshot%202026-03-10%20113546.png)

Per the [site](https://toolbox.googleapps.com/apps/messageheader/analyzeheader), "Messageheader analyzes SMTP message headers, which help identify the root cause of delivery delays. You can detect misconfigured servers and mail-routing problems".

**Usage**: Copy and paste the entire email header and run the analysis tool.
